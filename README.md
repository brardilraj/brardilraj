## 🚀 Featured Projects

### 🌾 [AgriSense – AI Crop Yield Prediction](https://github.com/brardilraj/AgriSense-AI-Driven-Crop-Yield-Prediction)
> Full-stack ML platform predicting crop yield for Tamil Nadu using RandomForest on 17 soil/climate features

**Tech Stack:** `Python` `Flask` `Scikit-learn` `RandomForest` `Pandas` `Chart.js`

- 🎯 Achieved superior R² scores vs XGBoost & DNN in 5-fold cross-validation
- 🔗 Built REST API with real-time predictions, crop recommendations & district rankings
- 📊 Interactive dashboard with Chart.js visualizations & animated KPI cards
- 🔒 Bandit security audit & AWS-ready architecture

---

### 🛡️ [Deep Learning Content Moderation](https://github.com/brardilraj/MultiLabel-Toxicity-Moderation)
> Bi-LSTM model classifying toxic comments across 6 simultaneous categories

**Tech Stack:** `Python` `TensorFlow` `Keras` `Bi-LSTM` `NLP`

- 🏆 **0.9766 ROC-AUC** & **0.6358 Macro F1** — 35% better than baselines
- 🧠 2.8M parameter Bidirectional LSTM with 128-dim embeddings
- ⚖️ Handled extreme class imbalance (143K:478 ratio) using semantic embeddings
- 📈 Trained on Kaggle's Jigsaw Toxic Comment dataset (153K+ rows)

---

### 🎨 [HR Workflow Designer](https://github.com/brardilraj/HR-Workflow-Designer)
> Production-grade visual workflow editor for HR automation with drag-and-drop canvas

**Tech Stack:** `React 19` `TypeScript` `Zustand` `React Flow` `Tailwind CSS`

- 🎯 5 custom node types with DAG traversal engine for real-time simulation
- 🏗️ Type-safe architecture with Zustand state management & useMemo optimization
- 🌗 Dark/light theme system with CSS variables & localStorage persistence
- 🔄 Service layer abstraction enabling seamless backend integration swap

---

### ⚽ [Tactifit – AI Football Scouting](https://github.com/brardilraj/tactifit)
> Data-driven player recommendation system with web scraping & clustering

**Tech Stack:** `Python` `FastAPI` `Scikit-learn` `Selenium` `K-Means` `PCA`

- 🕷️ Real-time web scraping with Selenium & BeautifulSoup
- 🎯 PCA + K-Means clustering for performance-based player grouping
- 🔍 Similarity-based recommendation engine with FastAPI REST endpoints
- 📊 Budget, age, and position-constrained player discovery
